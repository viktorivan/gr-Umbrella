    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1><a id="user-content-gr-umbrella" class="anchor" href="#gr-umbrella" aria-hidden="true"><span class="octicon octicon-link"></span></a>gr-Umbrella</h1>

<p>An out-of-tree module for GNU Radio with IT++.</p>

<p>Prerequisites:</p>

<p>GNU Radio IT++</p>

<p>See these steps fow a quick build guide.</p>

<p>(1) Ensure that you've satisfied the external dependencies. These dependencies are GNU Radio and IT++.</p>

<pre><code>See the wiki at http://gnuradio.org for GNU Radio details
and http://itpp.sourceforge.net/stable/ for IT++ details.
</code></pre>

<p>(2) Building from cmake on root folder:</p>

<pre><code>$ mkdir build
$ cd build
$ cmake ../
$ make
$ sudo make install
$ sudo ldconfig
</code></pre>

<p>Enjoy it!!!</p>
